GREEN GARDEN SYSTEM 1.0

Diseñado para la adminitración de jardines botanicos

¡Ayuda!

Esta pagina web esta creada con la opción de poder brindar ayuda al usuario que use dicho programa (Green Garden System).

PRINCIPAL

“Si los sentimientos fueran plantas, tú serías el jardín botánico.”

La pagina principal consta de 13 botones los cuales estan especificados cada uno por su nombre entre estos tenemos usuarios, visitantes, plantas, vivero, membresias, empleados, zonas, mantenimiento, reportes, ayuda, acerca de, cerrar sesion y salir. Cada uno tiene las distintas especificaciones para lo que fueron creados.

USUARIOS

“Si los sentimientos fueran plantas, tú serías el jardín botánico.”

Este botón indica que ahí se encuentran los formularios de “usuarios” dentro de el, habrá un sub menú indicando dos opciones que son: usuarios y tipo de usuarios. Aquí solo puede agregar datos el administrador, ya que el tiene el poder absoluto del programa. Es necesario llenar primero tipo de usuarios.

VISITANTES

“Si los sentimientos fueran plantas, tú serías el jardín botánico.”

Este botón indica que ahí se encuentran los formularios de “Visitantes” dentro de el, habrá un sub menú indicando dos botones los cuales son: visitantes y tipo visitante. En esta opción solo el administrador y el empleado podran agregar datos. Es necesario llenar primero tipo visitante.

PLANTAS

“Si los sentimientos fueran plantas, tú serías el jardín botánico.”

Este botón indica que ahí se encuentran los formularios de “Plantas” dentro de el, habrá un sub menú indicando 7 formularios, los cuales son: region, tipo, uso, genero, especie, familia, plantas. En esta opción solo el administrador y el empleado podran agregar datos. Es necesario llenar todos los demas formularios excepto plantas.

VIVERO

“Si los sentimientos fueran plantas, tú serías el jardín botánico.”

Este botón indica que ahí se encuentran los formularios de “Vivero” dentro de el, habrá un sub menú indicando 3 formularios, los cuales son: producto, tipo producto, ventas. En esta opción solo el administrador y el empleado del vivero podran agregar datos. Es necesario llenar primero tipo producto.

MEMBRESIAS

“Si los sentimientos fueran plantas, tú serías el jardín botánico.”

Este botón indica que ahí se encuentran los formularios de “Membrecías” dentro de el, habrá un sub menú indicando 2 formularios, los cuales son: tipo y afiliados. En esta opción solo el administrador y el empleado podran agregar datos. Es necesario llenar primero tipo.

EMPLEADOS

“Si los sentimientos fueran plantas, tú serías el jardín botánico.”

Este botón indica que ahí se encuentran los formularios de “Empleados” dentro de el, habrá un sub menú indicando 2 formularios, los cuales son: empleados y cargo. En esta opción solo el administrador y el empleado podran agregar datos. Es necesario llenar primero cargo. Aquí se determina los empleados que estan tanto como en el jardin botanico como en el vivero.

ZONAS

“Si los sentimientos fueran plantas, tú serías el jardín botánico.”

Este botón indica que ahí se encuentran los formularios de “Zonas” dentro de el, habrá un sub menú indicando 5 formularios. Aquí se muestran cuales se tienen que llenar de primero hasta el último. En esta opción solo el administrador y el empleado podran agregar datos. Es necesario llenar primero tipo.

MANTENIMIENTO

“Si los sentimientos fueran plantas, tú serías el jardín botánico.”

Este botón indica que ahí se encuentran los formularios de “Mantenimiento” dentro de el, habrá un sub menú indicando 3 formularios, los cuales son: mantenimiento, empleados, tipo. En esta opción solo el administrador y el empleado podran agregar datos. Es necesario llenar primero tipo.

REPORTES

“Si los sentimientos fueran plantas, tú serías el jardín botánico.”

Este botón indica que ahí se encuentran los formularios de “Reportes” dentro de el, habrá un sub menú indicando 9 formularios. Aquí se mostraran todos los reportes necesarios a la hora de necesitarlos cada mes.

AYUDA

“Si los sentimientos fueran plantas, tú serías el jardín botánico.”

Este botón indica lo que es esta pagina web.

ACERCA DE

“Si los sentimientos fueran plantas, tú serías el jardín botánico.”

Este botón indica que ahí se encuentran los formularios de “Acerca de” dentro de el, se da a conocer la información acerca de nuestro programa.

CERRAR SESION

“Si los sentimientos fueran plantas, tú serías el jardín botánico.”

Este botón indica que se podrá cerrar sesión de usuario.

SALIR

“Si los sentimientos fueran plantas, tú serías el jardín botánico.”

Este botón indica que se podrá salir del sistema de una forma segura.
